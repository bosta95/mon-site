Guide d'Installation IPTV

Découvrez comment configurer facilement votre service IPTV

Ce guide d'installation vous accompagne pas à pas pour configurer IPTV Smarters Pro sur votre appareil. Suivez ces étapes simples pour accéder à plus de 22 000 chaînes, films et séries en haute définition.

Tutoriel vidéo explicatif

Pour une démonstration complète de l'installation et de la configuration d'IPTV Smarters Pro.

1

Télécharger l'application Downloader

Pour installer IPTV Smarters Pro sur votre Fire TV, vous aurez besoin de l'application Downloader. Ouvrez votre bibliothèque d'applications (ou l'Appstore Amazon) et recherchez "Downloader". Sélectionnez l'application et procédez au téléchargement.

2

Autoriser les sources inconnues

Avant de pouvoir installer IPTV Smarters Pro via Downloader, vous devez autoriser l'installation d'applications de sources inconnues sur votre Fire TV. Pour cela :

  • Accédez à Paramètres > Ma Fire TV > Options de développeur.
  • Activez Applications de sources inconnues pour l'application Downloader.
3

Installer IPTV Smarters Pro via Downloader

Lancez Downloader, puis :

  1. Dans la barre d'adresse, saisissez le code direct du fichier APK d'IPTV Smarters Pro (164547).
  2. Appuyez sur Go ; patientez 6 secondes. Si le téléchargement ne démarre pas, cliquez sur le lien.
  3. Une fois le téléchargement terminé, suivez les instructions à l'écran pour installer l'application.
4

Ouvrir et configurer IPTV Smarters Pro

Après l'installation, ouvrez IPTV Smarters Pro. Vous serez invité à accepter les conditions d'utilisation et à accorder les autorisations nécessaires (accès au stockage, etc.). Sélectionnez ensuite « Login with Xtream Codes API ».

5

Ajouter votre abonnement IPTV

Saisissez les informations fournies par votre fournisseur (URL, nom d'utilisateur, mot de passe, etc.). Assurez-vous de bien vérifier les informations avant de valider pour éviter tout problème de connexion.

6

Personnaliser l'interface

Dans les paramètres, vous pouvez personnaliser l'interface, trier vos chaînes, bloquer certaines catégories, gérer vos favoris, ou encore activer l'option EPG (Guide TV) si elle est incluse dans votre abonnement.

7

Profiter de votre abonnement

C'est terminé ! Vous pouvez maintenant profiter de votre abonnement IPTV Smarters Pro, parcourir vos chaînes, films et séries préférés, et bénéficier d'un streaming de haute qualité. N'hésitez pas à explorer toutes les fonctionnalités disponibles.

Questions fréquentes

Installation sur Android

Pour installer sur un smartphone Android, téléchargez simplement l'application depuis le Google Play Store, puis suivez les étapes 4 à 7 de ce guide.

Compatibilité iOS

L'application est disponible sur l'App Store d'Apple pour iPhone et iPad. La configuration suit les mêmes principes que ce guide.

Problèmes de lecture

Si la lecture est saccadée, vérifiez votre connexion (10 Mbps min.), rapprochez-vous du routeur ou réduisez la qualité vidéo dans les paramètres.

Gestion des favoris

Pour ajouter aux favoris, appuyez longuement sur OK pendant la lecture ou sur l'icône en forme d'étoile dans la liste des chaînes.
